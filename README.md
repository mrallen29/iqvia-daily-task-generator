\# Prod\_task\_generator



A desktop tool for generating Start‑of‑Day (SOD), End‑of‑Day (EOD), and OT email drafts.



\## Features

\- SOD / EOD / OT email draft generation

\- Task presets

\- Outlook Classic and New Outlook support

\- Built‑in auto‑update via GitHub Releases



\## Installation

Download the latest `Prod\_task\_generator.exe` from the Releases page and run it.



\## Updates

The app automatically checks for new versions using GitHub Releases.



\## Notes

This repository contains the \*\*source code only\*\*.  

The compiled `.exe` is distributed via GitHub Releases.
